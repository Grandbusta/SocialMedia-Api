## Docs

### Users endpoints

Login User:

```
POST users/signin
```

Create a new user:

```
POST users/signup
```

Delete User:

```
DELETE users/delete
```

Update User:

```
PATCH users/update
```

Get all Users Posts:

```
GET /users/:userID/posts
```

Get all Users Friends:

```
GET /users/:userID/friends
```
